---------------------------------------------------------
---------- 开源代码生成器
----------  At  2015.8.28   By 陈良
--------------------------------------------------------
目的：解救加班程序猿，自动化代码生成器
思路：把需要生成的重复生成的源代码代码按照指定规则编辑好  放在 代码模板文件夹(code_template)，
           设定代码生成规则；
           生成你想要的源代码(code_target)
           
项目文件命名约定

目录约定：
src                   源文件
code_target       生成的目标文件目录
code_template   源文件模板文件夹
libs                    项目所需的jar包文件

包命名约定：
源码包主目录: com.opensource.maker 
 .input 输入部分代码
 .output 输出部分代码
 .main 项目启动，控制部分代码
 .test   测试代码
 .common 公共部分代码
 
 
 -------------------------------------------------------
 程序约定：
 1.目标目录 和 模板文件目录说明
 1.1 放在comm 目录中的所有文件都会原封不动copy过去
